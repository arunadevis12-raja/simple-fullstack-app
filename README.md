# Simple Fullstack App (Node.js + React)

This is a beginner-friendly full stack project built using:

- Node.js
- Express.js
- React.js

## Features
- User Registration API
- User Login API
- React frontend

## Tech Stack
- Backend: Node.js, Express
- Frontend: React

## How to Run Locally

### Backend
```bash
cd backend
npm install
node index.js


### Run Frontend

```bash
cd frontend
npm install
npm start
